# Selenium Automation Framework

## Overview

This project is a Selenium Automation Framework built using Java, TestNG, Maven, and the Page Object Model (POM). It is designed for scalable and maintainable web automation testing.

---

## Tech Stack

- Java 17
- Selenium 4
- TestNG
- Maven
- Page Object Model
- Extent Reports
- Git
- GitHub
- Jenkins
- GitHub Actions
